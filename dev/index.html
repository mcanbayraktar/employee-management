üìÅ Component Architecture

The application is built with three main components:

1. Employee List Component (employee-list.js)

  • Displays employees in a responsive card grid
  • Search and filter functionality
  • Employee statistics and empty states
  • Edit and delete actions

2. Employee Form Component (employee-form.js)

  • Modal-based form for creating/editing employees
  • Real-time form validation
  • Accessible form controls
  • Responsive layout adaptation

3. Main App Component (employee-app.js)

  • Application state management
  • Component orchestration
  • Statistics calculation
  • Local storage integration
